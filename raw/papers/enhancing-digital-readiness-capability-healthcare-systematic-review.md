# Enhancing Digital Readiness and Capability in Healthcare: A Systematic Review of Interventions, Barriers, and Facilitators

Alotaibi, N., Brown Wilson, C., & Traynor, M. (2025). BMC Health Services Research, 25:500.

## 摘要

本系統性回顧旨在識別提升醫療專業人員數位準備度（digital readiness）和數位能力（digital capability）的介入措施，並理解其在數位轉型過程中遭遇的障礙與促進因素。

- **方法：** 混合方法系統性回顧，依循 Joanna Briggs Institute (JBI) 指南。搜索 CINAHL Plus、MEDLINE、EMBASE、PsychINFO、Web of Science 五個資料庫。使用 UTAUT 框架分析技術採用因素。
- **結果：** 初始識別 1140 篇研究，篩選後納入 21 篇（2017–2023 年發表）。12 篇質性研究、4 篇量化研究、5 篇混合方法研究。
- **結論：** 提升績效期望、解決努力期望、改善促進條件、善用社會影響力，是成功推動數位健康採用的關鍵。

## 理論框架：UTAUT

本研究使用 **Unified Theory of Acceptance and Use of Technology (UTAUT)** 框架，由 Venkatesh et al. (2003) 整合八大理論模型而成，解釋力達 69%（調整後 R²），優於各單一模型（17%–53%）。

### UTAUT 四大核心構面

| 構面 | 定義 |
|------|------|
| **Performance Expectancy (PE)** | 個人相信使用該系統能提升工作績效的程度 |
| **Effort Expectancy (EE)** | 使用系統的容易程度 |
| **Social Influence (SI)** | 個人感受到他人認為其應使用新系統的重要程度 |
| **Facilitating Conditions (FC)** | 個人相信組織和技術基礎設施能支持系統使用的程度 |

調節變項：性別、年齡、經驗、自願性。

## 主要發現

### Theme 1：Performance Expectancy（績效期望）

七篇研究探討此構面。關鍵發現：

- 前線醫療工作者認為**感知有用性**（perceived usefulness）是接受技術的關鍵驅動力
- 護理師對護理資訊系統的使用意向受績效期望顯著影響
- **能力和動機**均對數位醫療的採用行為意向有正面影響
- 在接受型文化（如中國），**操作容易度**比績效期望更重要
- 熟悉線上經驗不影響數位工具的感知和採用；但高數位素養則有正面影響

### Theme 2：Effort Expectancy（努力期望）

三篇研究聚焦此構面：

- 數位工具的**使用容易度**是技術接受的重要因素
- **複雜性和多次登入**是顯著障礙
- 多重登入和登出需求導致生產力損失
- 資訊素養對努力期望有正面影響

### Theme 3：Social Influence（社會影響）

兩篇研究涉及此構面：

- 激勵醫療工作者並闡明數位平台的附加價值
- **共同決策**和以病人為中心的照護在建立感知價值、激勵團隊擁抱技術方面扮演重要角色
- 清晰的願景和商業案例有助於推動採用
- 教育患者和醫療提供者如何使用數位工具

### Theme 4：Facilitating Conditions（促進條件）

最多研究涉及的構面，分為兩個子主題：

#### 子主題 4a：Insufficient Training（訓練不足）

八篇研究識別訓練不足為顯著障礙：

- 數位健康問題解決訓練不足 → 基本數位能力低落
- 多數醫療專業人員缺乏基本硬體和軟體技術技能
- **性別**：男性數位能力高出 3.9 倍
- **教育程度**與數位能力正相關
- **年輕專業人員**對工作環境變化更具接受度
- 專業教育中接觸數位健康的機會有限
- 訓練不足和缺乏在職支持 → 壓力和無能感 → 抗拒 IT

#### 子主題 4b：Organizational and Technical Infrastructure（組織與技術基礎設施）

五篇研究強調組織和技術基礎設施的重要性：

- 硬體故障、軟體缺陷、供電不穩 → 影響使用行為
- 需要提供數位能力分享的資源和機會，營造友善安全的數位組織氛圍
- **隱私與安全疑慮**顯著影響技術接受度（62.5% 參與者同意技術可能侵犯隱私）
- 對安全和隱私的負面感知 → 降低技術接受態度
- 醫療工作者對數位健康工具數據準確性的疑慮

## 障礙摘要

| 障礙類型 | 具體障礙 |
|---------|---------|
| **訓練不足** | 問題解決訓練缺乏、專業教育中數位健康教育不足 |
| **基礎設施** | 硬體故障、軟體缺陷、供電不穩、電腦不足 |
| **可用性** | 多重登入、系統複雜性、自動登出 |
| **抗拒變化** | 偏好傳統方法、對新技術猶豫 |
| **隱私與安全** | 數據隱私疑慮、資訊安全信任不足 |
| **技術技能** | 基本數位能力不足、數位素養差距 |

## 促進因素摘要

| 促進因素 | 說明 |
|---------|------|
| **感知有用性** | 相信技術能提升工作績效和病人照護 |
| **專業訓練** | 整合數位健康模組到學程和持續專業發展 |
| **組織支持** | 技術支援、充足資源、支持性環境 |
| **社會影響** | 同儕激勵、共同決策、以患者為中心照護 |
| **經驗** | 高數位素養和先前數位經驗有正面影響 |
| **動機和能力** | 均對數位健康採用的行為意向有正面影響 |

## 研究方法

- **類型：** Mixed-methods systematic review
- **指南：** JBI Manual for Evidence Synthesis
- **品質評估：** Mixed Methods Appraisal Tool (MMAT, 2018)
- **註冊：** PROSPERO (CRD42023461309)
- **報告標準：** PRISMA 2020
- **資料庫：** CINAHL Plus, MEDLINE, EMBASE, PsychINFO, Web of Science
- **納入標準（PEO）：** 醫療專業人員 × 數位能力/準備度/素養 × 介入措施和障礙促進因素
- **排除：** 學生、患者、非醫療職業、資訊素養、健康素養、e-learning、模擬學習
- **篩選流程：** 1140 → 去重後 1083 → 標題摘要篩選後 183 → 全文篩選後 28 → 品質排除低品質 7 篇 → 最終 21 篇

## 實務建議

1. 將專門數位健康模組整合到**學程課程**和持續專業發展計畫中
2. 依不同數位素養程度**客製化訓練**
3. 投資建設支持性的**技術基礎設施**
4. 建立清晰的**組織願景和商業案例**以推動數位採用
5. 善用**社會影響力**（同儕激勵、共同決策）

## 研究限制

- 僅搜索英文文獻
- 資料庫最後搜索日期為 2023 年 4 月 29 日
- 僅使用 UTAUT 單一理論框架
- 聚焦範圍限於數位健康知識、能力框架、數位素養等特定主題

## 參考文獻

包含 91 篇參考文獻，涵蓋數位健康、技術接受模型、醫療資訊系統、數位能力框架等領域。
